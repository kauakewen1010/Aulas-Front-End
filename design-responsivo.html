Design Responsivo ( ou responsibilidade)

Conjunto de técnicas de desenvolvimento front-end visando a construção de sites em que o layout e a experiência geral de uso funcionarem bem em dispositivos com telas em tamanhos variados.

Existem duas abordagens para o desenvolvimento responsivo:

Mobile First (é o mais usado)

Nesta abordagem o desenvolvimento é focado primeiro em telas pequenas, para somente depois realizarmos ajustes para telas maiores.

Desktop First

Nesta abordagem o desenvolvimento é focado primeiro em telas grandes, para somente depois realizarmos ajustes para telas pequenas.
